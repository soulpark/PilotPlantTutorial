웹 페이지 로딩하기

레퍼런스를 확인할 수 있는 샘플 프로젝트를 제공해 드리고 있습니다. 샘플 프로젝트를 이용해 기능과 동작을 확인하신 후 직접 만들어 보시기 바랍니다.

웹 뷰는 웹 페이지를 로드할 수 있는 뷰 입니다. CHWebViewController를 이용하면 프로토타입 구동 중 원하는 웹 페이지나 pdf 파일을 화면에 표시할 수 있습니다.

실행할 스토리보드 파일을 선택합니다. 프로젝트 타겟의 General 설정사항 중 Main Interface에 WebView 스토리보드를 선택합니다.

WebView 스토리보드를 선택합니다.


뷰 컨트롤러의 이름을 CHWebContentViewController로 변경한 뒤, User Defined Runtime Attributes 항목에 'URL' 키를 만들고 원하는 웹 주소를 입력합니다.

CHWebContentViewController에 URL을 입력합니다.


Run 버튼을 눌러 실행해 보면, 지정한 웹 페이지가 로딩되는 것을 확인할 수 있습니다.

codershigh.com 웹 페이지가 로드 되었습니다.


pdf 파일 로딩하기

뷰 컨트롤러의 이름을 CHWebContentViewController로 변경한 뒤, User Defined Runtime Attributes 항목에 'fileNameExt' 키를 만들고 원하는 파일의 이름을 입력합니다. 파일은 프로젝트에 추가되어 있어야 합니다.

제공되는 샘플 프로젝트에서 첫 화면 화살표를 두번째 씬으로 옮겨서 바로 실행해 볼 수 있습니다.

CHWebContentViewController에 파일이름과 확장자를 입력합니다.


Run 버튼을 눌러 실행해 보면, 지정한 pdf 파일이 로딩되는 것을 확인할 수 있습니다.

선택한 pdf 파일이 로드 되었습니다.
